In addition to participating as a developer in the UMN API project, I have created an identity and website for the group to use for public-facing work and web apps.
The mark is intended to evoke two ideas: graph diagrams frequently used in describing algorithms and computer systems, and far-flung University areas being centralized.
The site is designed to position the group among other new, web-savvy projects with a focus on very clean divisions between visual areas and strong hierarchy.
